use github to save your code

发表于 2018-01-12

首先在网页上新建项目：new Repository

然后在本地： 

    git clone https://github.com/lukezhg/space-craft.git
    cd space-craft
    git add .
    git commit  -m "save space craft"
    git push -u origin master #需要输入用户名和密码
