能用Scribefire撰写的国内博客

发表于 2017-03-24 更新于 2013-01-11 分类于 Firefox

Scribefire是一个能够在线撰写博客的火狐浏览器插件，目前发现新浪和网易都支持windows live writer，同时也支持scribefire，但是scribefire打开的时候都存在分类不能识别的问题。

下满是网易和新浪的api地址：
1、网易：http://os.blog.163.com/api/xmlrpc/metaweblog/ 
2、新浪：http://upload.move.blog.sina.com.cn/blog_rebuild/blog/xmlrpc.php
