用flash制作试题交互动画项目

发表于 2017-03-24 更新于 2013-01-11 分类于 技术技巧 ， flash

在网络中有很多互动的练习题都是用flash制作的，互动效果很好，学生可以利用这些动画巩固学习的知识，检测自己的掌握程度。但是很多练习题都是不提供源文件的，即便提供了原文件往往试题数据和源文件并没有分离，教师很难更改并为我所用。那么是不是可以创建一个flash源文件，让稍微动flash甚至是一点都不同flash的老师也能修改并展示自己的试题呢？当然是可以的，但是这个试题交互动画的设计应该数据与源文件分离的，也就是老师不需要修改源文件和代码，只需要修改数据文件就可以。

制作的思路：
1、建立界面fla原文件（test.fla）
2、建立代码文件(xmlloader.as)
3、建立试题数据xml文件(sample.xml)

使用代码加载试题数据xml文件，并在动态文本中展示试题，学习者点击选项按钮以及导航按钮参与互动。互动可以分为两种模式，一种是练习模式，一种是测验模式，用户可以自己选择。此外为了使得该动画可以在网站上调用，还增加一个输入文本框，用于导入特定位置的试题数据。

发布：
最有作品提供全部文件，并以CC协议中的（署名 非商业使用 相同方式共享） 发布。

从宫廷剧能看到什么

发表于 2017-03-24 更新于 2013-02-18 分类于 生活

宫廷剧一部接着一部，一集接着一集，大有主导电视剧市场的趋势，这意味着什么？

最近看了几集《甄嬛传》，只是看了开头，中间几集以及结尾，70多集每集都看实在是看不过来。电视剧的情节很是吸引人，演员也不是演的不好，但是静下心来想一想，宫廷剧这种封建社会畸形人际关系实在是看多无益。而且这种封建的君臣思想以及后宫的明争暗斗都让人感到压抑、冷漠，不知道观众看到的是“皇帝的龌龊”还是“智慧”，是甄嬛的“顺应”还是“抗争”，总是觉得宫廷剧题材实在是憋闷，角色无非是皇帝、皇宫的女人、太监、医生，就像八股文，演来演去还是那些争斗，人性的负面体现的淋漓尽致，但主体性、创造性荡然无存。

如今的中国应该说正处于剧烈的社会经济变革之中，很多社会经济问题都是非常好的电视剧、电影题材，但是为什么没有更多的拍出来呢？

互联网上的榜样值得我们思考和借鉴

发表于 2017-03-24 更新于 2013-01-11 分类于 反思 ， 学习社区

今天，我看到两件事，值得学习，所以在这里记录和分享一下。

第一件事，在使用google搜索“想法”这个关键字的时候，无意间发现一个很好的小视频。（改變一下自己的想法 - YouTube
: http://www.youtube.com/watch?v=srnUwqPf-DU） 于是顺藤摸瓜发现作者的网站：倒著看人生-生命教育工程
: http://www.edu-for-life.com/

以及作者周凡的优酷视频空间: http://u.youku.com/user_show/uid_upsidedown98

反思：作者运用非常简洁的表达方式，文字+简笔画+音乐，甚至音乐都没有变过，但是却形成了非常有特色的风格，深入浅出，与人生智慧与浅显故事之中，每个视频仅有1-2分钟。这样的表发方式以及教育目标和传播手法都值得借鉴和反思。

第二件事，在打开我们社区主页（http://ecnubio.wikispaces.com/）的时候，看到了pplong分享的博文“与学生相处的四个技巧”，而且发现我们社区的成员已经阅读了这篇博文，在博文的下面有很多读者与博主在进行交流，博主一一对评论进行了回复。

反思：教育博客其实是一个很好的交流平台，如果维护好了不仅可以成为自己的品牌，同时也能有效的扩展思路，广交好友。博客群也可以较大的让新博主快速扩展人际圈子，加强互动交流。

总结：这两件事情是互联网上再普通不过的事情，但是只要我们留心，我们就会发现很多做事的人，做事的目的，做事的方式和方法，进而有所启发，有所感悟，然后付之行动。

实验教学理论课的教学反思

发表于 2017-03-24 更新于 2015-03-10 分类于 教学

今天上了实验教学的理论课，一个很深的体会是如果让学生连续听2个半小时的讲座，想让学生一直高度关注是很难的。反过来，让一个教室连续讲上2个半小时，想不累也是很难的。今天这次课就是同时做了这两件事情。

成功之处：
觉得一些好的例子，是很不错的。比如小学探究热量的案例，光合作用影响因素的案例。此外三盒米饭的例子也很不错。对于思想的形象化解释和比喻也是不错的。在讲解的过程中调动学生的已学知识和概念的做法也是好的。

不足之处：
总是想让学生动起来，不仅仅是行动起来，同时也是思考起来，活跃起来，更愿意表达核分享，但是同学还是不是十分想敢说。有啥说啥的勇气还是缺乏。
另外想讲的内容很多，而且ppt没有大标题标记，显得有些混乱，标题和标题之间的线索有些不清晰了。z老师提出建议其实可以把所有内容概括为三大块，做好大标题标记，并去掉一些多余的案例和干巴巴的理论，会更好。这些建议是中肯的。
在语言表的基本功上，我的语言表达能力还需要练习，表达得特别快的时候就会找不到合适的词语或者吐字不清，这点其实要在平时多积累词汇，多写，多说，厚积薄发，自然水到渠成。

突发事件：
刚开始想使用投影仪，但是发现怎么弄都投不上去，于是换了一个接口，头上去了。
讲授的时候发现投影仪接口与前面的讲台很远，于是自己搬了个凳子在加了一个箱子做了一个临时讲台，效果不错。这样可以看电脑上的ppt也可以看到同学的眼睛了。（但是还是应该考虑买一个ppt遥控器）
本来想使用ofiice2007的演讲者播放模式，这样可以看自己的笔记同时大屏幕只显示演示内容，但是弄了一些没成功。
分组的时候，有两个组一个组全是女生，一个组全是男生，为了男女搭配，我只好让他们重新组合一下，但是有一点强制性了，对不住这两位女同学了，把她们调过去的时候她们可能不是特别乐意。现在想来还是可以让他们私下商量解决的，不必要教师干预。毕竟他们都是成人了。

学生见解：
有同学提出想借用一楼的微格教室，但是为了管理方便，还是建议他们使用实验教室。但是提前就多练习和操练微格的这个意识是值得充分肯定的，这也说明有些同学确实是非常主动的。

宝宝跳跳器——脚手架

发表于 2017-03-24 更新于 2013-02-18 分类于 生活



“跳跳器是一种悬挂在门框金属夹具上的塑料椅或布椅。它由链条、弹簧或橡胶和布带构成。宝宝在上面跳起，就可以弹离地面。一些宝宝不喜欢跳跳器，而其他宝宝却很喜爱，直到学会走路。

跳 跳器也有一些问题：夹具可能会松开，带子可能会磨断，从而导致宝宝掉到地上或撞到门框。一些座椅对宝宝的大腿内侧压力过大，导致瘀痕或影响血液循环。安装 跳跳器也具有一定的难度。宝宝长时间使用会导致晕眩，症状与晕船相似。同时还有鞭抽式损伤的危险，因此我们不提倡使用跳跳器。” 摘自：怎样为刚刚学走路的孩子选择用具？ 博闻网: http://health.bowenwang.com.cn/how-to-choose-baby-equipment8.htm

今天偶然在路上可到有一个家庭正在给宝宝使用跳跳器，这让我联想到脚手架的原理。宝宝在还没有学会走路的时候，家长有时候会用双手架在宝宝的腋下帮助其承担一部分的重量，这样宝宝就可以在双腿力量上不足够支撑身体重量的时候学习走路，这更我们的脚手架原理是一样的，学习者在教师提供特定帮助下完成学习任务，然后这种帮助会慢慢减少，直到学习者可以独立完成任务。

这正应了那句名言“教是为了不教”。
还有一句是“不教也是一种教”。

在iphone4 5.1.1上也能安装siri

发表于 2017-03-24 更新于 2013-08-25 分类于 技术技巧

昨天重新安装siri时悲剧了，变成了白苹果，后来刷了机。

刷到5.1.1系统之后还是忍不住在iphone上装了siri，具体的方法见：http://blog.sina.com.cn/s/blog_517625cc01012whj.html

1. 升级iTunes到最新
2. 备份iPhone 4
3. 下载absinthe 2.0.4 , 以管理员方式运行，破解完成，出现Cydia
4. 恢复备份
5. 在Cydia里添加源 repo.siriport.ru,
然后寻找SiriPort(original)安装之。然后重启IpHONE4.
然后在SAFARI里输入siriport.ru/siriport_ru_original.pem，安装证书。然后再次重启。
6. 在Settings-General-Enable SIRI启动SIRI。ENJOY!
人啊，总是造成了严重后果的时候，再会后悔不该当初，才知道节制和让步。
