什么是开放学习

发表于 2011-11-17 更新于 2013-01-11 分类于 openlearning

目前，网络上已经有很多学习资源了，人们称之为开放教育资源（OER）。
但是开放教育不仅限于资源的开放，还应包括教育过程的开放，以及学习过程的开放。因此除了开放课程，公开课，作为学习者，我们也应该开放的进行学习，并把自己的学习过程也开放，为更多的学习者提供线索。

那么什么是开放学习呢？

  • 开放学习是学习者依照自己的兴趣独立自主的学习。
显然，开放学习不仅需要学习者具有很强的自我调节能力，还需要一个支持性的外部学习环境。而这样的环境需要每个开放学习者的努力去营造，为此开放学习者要学会：
  1. 记录并分享整个学习目的、学习过程和学习成果
  2. 在学习过程中善于利用他人的学习经验以及集体智慧
  3. 有意识的创造和维护一个良好的学习环境

如何把tiddlywiki笔记导入到evernote

发表于 2011-11-11 更新于 2013-01-11

tiddlywiki是一个很不错的非线性网页笔记本，它是一个存在你硬盘上的一个单个html文件，但是打开之后你会发现它实际上是一个用语言写的笔记本程序，而且你可以更具自己的需要进行界面以及插件的定制，比如我为学生定制的tiddlywiki(http://ecnubio.wikispaces.com/tiddlywiki)。

在tiddlywiki已经有了很多笔记之后，它的打开和保存速度会有所下降，而且我使用了很多笔记服务，我想搜索笔记的时候经常忘记，这个笔记是在博客上，还是在diigo上，还是在googlereader上，还是在tiddlywiki上，还是在evernote上。所以你看，我的问题很明显，我要要把所有的笔记最好能在同一个地方存储一份。（这个问题让我很头疼，若果有其他更好的解决方法，不妨赐教）

所以我要把tiddlywiki上的笔记导入到evernote以便备份和搜索。尝试的解决方案：

  1. 使用ExportTiddlersPlugin: (http://www.tiddlytools.com/#ExportTiddlersPlugin)导出，纯文本文件，然后再编写文本分割程序，将每个笔记存为单独的txt文件。然后使用evernote-工具-导入文件夹。但是遇到了问题，第一导出的纯文本非常混乱，而且中文出现了乱码。折腾了半天，这个方案弃用。
  2. 又重新搜索，于是找到了一个老外的解决方案（The Strange Zen Of JavaScript: TiddlyWiki hack: exporting tiddler HTML: http://jszen.blogspot.com/2005/06/tiddlywiki-hack-exporting-tiddler-html.html）。在这个方案里提到使用火狐浏览器，可以通过选择tiddlywiki中的所有文字，然后右键，参看所选源代码，果然我看到了tiddlywiki中的所有笔记，而且这个笔记的格式非常统一，于是就动手把相应的笔记拷贝到一个txt文件中。接下来使用py编写了一个简单的文本切割程序，把文件中每个笔记单独的保存到一个txt文件中，然后导入，大功告成。yeah！！
总结：
  1. 动手之前好好做一下搜索工作非常必要，还是那句话，你遇到的问题99%已经有人遇到了过了，参考使用别人的解决方案，可以避免无谓的时间浪费和重复性工作。
  2. 遇到不熟悉的领域，找到相关的论坛进行搜索和求助是更容易的事情。
  3. 学好英文是非常关键的，这决定了你的阅读范围。能看英文的文档能让你的阅读范围扩大很多倍。多少倍呢？个人保守估计10倍左右，这取决于互联[[网上不用语言资料所占的比例]]。

教师应该是一个善于讲故事的人

发表于 2011-10-29 更新于 2013-01-11

很小的时候，我就很喜欢听故事。等大了一点，还编一些非常离奇的故事，将给小孩子听，虽然是现编现讲，而且非常离谱，但是它们居然还听的津津有味。这说明不了我多美会讲故事，只是说明孩子确实是非常喜欢听故事的。为什么呢？我觉得故事最贴近我们的生活体验，它是按照我们的个人经验范式叙事的，同时它也可能刚好符合了我们大脑的神经网络的情景记忆模式。

作为老师，要把一堂课上好，要用好的例子，比喻，表征，还有就是要会讲故事。无论是自己亲身经历的故事，还是别人的故事。善于搞科研的老师讲科研的故事，善于实践和反思的老师讲自己的故事，广泛阅读的老师讲经典的具有教育意义的故事。故事可以让你的课堂生动起来，鲜活起来，丰富起来，有趣起来，成为一个整体。当然前提是，你要善于讲，会讲故事。

教师讲故事应该把握一些原则：

  1. 主题相符（符合教学需要）
  2. 真实生动（真实可鉴，形象具体，绘声绘色）
  3. 短小精悍（占用时间短，语言精练）
  4. 发人深思（具有冲击性和启发性）
  5. 智慧活泼（让人会意一笑）
为了运用好这些故事，教师要注意日常的素材的搜集、整理和加工，最后才能为我所用。比如今天，我就在新世界数字出版网上看到一本在线读物《18岁以前孩子应该懂得的做事方法》（http://www.nwp.cn/book/233_6871.shtml），上面运用了很多故事素材，来达到教育意义。

教师也应该是一个有故事的人。平常的教学反思、教育叙事、生活经历都可以积累成自己的故事。

帮助学生提出他们自己的问题-跨上思想的野马

发表于 2011-10-17 更新于 2013-01-11

提问实际上是一个思考的过程？而提问是一种尝试建立连接的过程，因此对于学习非常有益。这种益处表现在，一方面，当你提问的时候，你会觉得寻找问题的答案是自己的责任，你会想方设法解决它。而当学生知道如何提出自己的问题时，他们会为自己的学习负起更多的责任，深化交流，建立新的连接和获得新的发现。

教师借助问题生成技巧（Question Formulation Technique）来辅助学生提问。

  1. 教师设计问题的焦点（图片、视频、声音，获得学会生注意，使其快速提出问题）
  2. 学生提出问题（规则是尽可能多的提出问题，不要停下来讨论，评判或者试图回答，写下原封不动的写下这些问题，把每个表述都变成问句。一但学生工作起来，这个规则可以使得学生进入开放的思考状态。）
  3. 学生完善自己的问题。区分哪些是开放性的，哪些是封闭性的问题。它们的优缺点如何，教师可以让学生至少把一个开放性问题转变成封闭性问题，并且把一个封闭性问题转变成一个开放性问题。让学生意识到，如何表述一个问题，会影响问题的深度、质量以及获得信息的价值。
  4. 学生优选自己的问题。单元学习开始时，选择你最想扩展学习的问题。实验时选择三个可以检验的问题。
  5. 师生共同决定下一步行动。
  6. 学生反思自己学到了什么。

欢迎参加活动——学生评价大讨论

发表于 2011-10-04 更新于 2013-01-11

作为教师经常关注的问题是怎么教？也就是教学法的问题，但是教什么，为什么教，教得怎么样？这样的问题却相对较少的关注。为什么呢，教什么已经由课程内容 标准以及教科书做了规定和一定的限制，所以教师关注与否都很难改变，而为什么教，也就是教育目标的问题，也已经有国家政策文件进行了阐述。至于教的怎么样 呢？看高考。所以留给教师的只有一个问题，那就是如何教，也仅仅在这个问题上一线教师才有话语权。

真的是这样吗？差不多。真的应该这样吗？我个人觉得绝不应这样。因为教师不等于教书匠。教师的工作是创造性的工作，而绝不是教育生产线上的终端分装器，把 各科知识按照需要分装到学生的头脑里。课程试图屏蔽教师个人的影响，就像绘画试图剔除画家特有的风格一样愚蠢。除非这个课程是没有教师的课程，但是即便如 此，也会涌现出一些学习领袖对整个课程的学习产生巨大的影响。学习领袖难道不是扮演了教师的角色吗。

因此，教师越不应该只关注怎么教，还要关注为什么教，教什么，教的怎么样，在哪里教，交给谁等一系列的问题，也只有这样教师才能成长为教学专家，成长为教育家。

而实际上为什么教，教什么，以及教得怎么样，这三个问题是密切相关，而且都会反映在学生评价当中。那么我们不妨就“学习评价”，就我们应该如何评价我们的 学生，这个问题展开一次讨论。考高命题也许并不由我们教师决定，但是课堂教学中对学生的评价的权利却在掌握在我们教师手里。我们要如何行使这项权利呢？ 欢迎大家参与此次在线讨论！

解决方案：求助机房管理人员安装必要软件和浏览器；鼓励学员使用便携版系统和浏览器。比如可以使用便携版火狐 Firefox Portable（新浪下载页面： http://down.tech.sina.com.cn/content/1528.html）

此外，很多开源或者绿色软件都可以尝试装在优盘中，而且不影响正常使用。

问题2：学员遇到问题之后，常常无不知道如何解决？
解决方案：

  1. 明确表述你的问题或者需要。比如问题1中由于机房电脑的限制，实际上我们需要的是找到能够在闪存中运行的版本的软件。
  2. 使用搜索引擎查找解决方案。比如上面的例子中，我只要搜索“便携版firefox”，结果就找到了先关的软件。
  3. 尝试按照方案解决实际问题。尝试是很重要的，如果你有了某些想法，不妨试一试，没准你就找到了一个绝佳的问题解决方案。没有尝试，就不能获得更多的反馈信心，没有这些反馈信息就根本找不到适合自己的方法。这就像学习游泳一样，教练把动作讲的再清楚，你不下水实践，那么还是不能学会如何游泳。
  4. 利用尝试过程信息求助论坛。经历了一些尝试之后，如果遇到了新的问题或困难，或者不知道进一步解决问题的方向，可以结合这些反馈信息到邮件列表或论坛里求助。这个步骤当然也包括向你认识的所有人求助，甚至包括不认识的相关专家。
  5. 记录分享你的问题解决方案。最后你可以把自己问题解决的过程以及最后的问题解决发难通过博客、wiki、邮件、微博等形式进行记录和分享，让在这个过程中给予你帮助的所有人也能分享你的解决方案。
问题3：如何更好的管理自己的学习过程？学员要以小组的形式完成教师选定的或者自己选定的学习任务，通常会有两个难题摆在学员面前：
  1. 如何提高团队的工作效率？
  2. 如何提高自己i的工作效率？
问题3.1：如何提高团队的工作效率？
解决方案：
  • 小组长要负起责任，做好组织和联系的工作，激发团队的士气以及互帮互助的氛围。
  • 明确的分工，必须把工作分解并分配每个人，当然最好让每个学员都能发挥自己的优势。
  • 设定截止日期，明确约定每个人在什么时候完成什么任务到什么程度。
  • 学会沟通和换位思考
  • 给每个学员以反馈和鼓励
问题3.2：如何提高个人的学习效率？
  • 使用电脑前，先列好清单，一件一件的完成，决绝网络其他娱乐方式的诱惑（比如聊天、浏览新闻、图片、视频等）。
  • 掌握好时间的分配。上课的时候应该专心听讲，而不应该在尝试去解决自己没有解决的问题，这些问题完全可以等老师讲完去问老师或同学。其次，在问题解决过程中如果遇到难题，10分钟还没有任何进展，暂时挂起该任务，另寻其他解决途径，而不应该花费一上午去查阅某些东西或者去尝试安装某些软件。因此在学习中，当一个任务10分钟还没有任何进展，那么就把这个问题挂起，或者抛到邮件列表里进行询问。另外要规划好自己的学习时间。
  • 最最重要的，最最本质的，起就是能够管理自己，自我调控。

不能引发学生兴趣的教学就像白水煮的肉

今天新疆骨干教师开学典礼，陆校长给我们打了一个比喻——白水煮的肉：

减负不是给教师减负，也不是简单的较少学生要学习的知识内容。假如学科内容像是一个块肉的话，过去我们的教师往往就是把肉煮了一下，就递给学生吃。没有任何作料的白肉能好吃吗？但是如果你减少肉的数量，那么学生就有可能吃不饱。所以我们教师应该做的是加一些作料，把肉做得好吃一些，让学生喜欢吃，保留学生对学科的一份兴趣，到了大学我们也好培养。
比喻很多时候能够触发我们对旧的事物产生新的思考，有关教学的比喻也是如此。我觉得陆校长的这个比喻就很有趣。教学就像做菜一样，教师就是厨师，学生就是顾客。如何做出一桌既可口有有营养的饭菜，却是对我们教师提出了挑战。但我觉得现在的教学不单单是趣味性的问题，也就是不单单是好吃不好吃的问题，而是营养不均衡。科学课程的内容不仅包括静态的知识，还包括获得知识的过程和方法，以及我们对科学本质的认识和情感。由于静态的知识相比较过程和情感而言更容易考察，更容易出题，出题的比重最大，又由于我们的学校、家长、老师都迷信考试或者都拜倒在了高考神像之下，以至于我们的教学充满了刻板的知识的传授，而缺乏活跃生动的能力锻炼和精神陶冶，就好像只给孩子吃含糖的食物，而没有蛋白质，结果虽然孩子吃得肚子涨涨的，但身体仍然缺乏营养，患了营养缺乏症。当然如果课程是烧菜原材料的话，那么教师教学就是厨师烧菜，虽然原材料中营养是平衡的，但是如果厨师在烧菜的过程中把某些营养损失了，又或者根本没有选择某种营养的材料，又或者都选择了却没有加调味剂，又或者加了调味剂却没有煮熟，学生都会营养不平衡。怎么办？一方面当然希望我们的老师成为好厨师，另外一方面则希望我们的老师教会学生自己做菜。

人是如何学习的

发表于 2011-08-29 更新于 2013-01-11

人类社会的进步和发展不仅表现在高楼大厦、电子数码、宇宙飞船，还表现在现代人的知识和技能水平。那么现代人为什么能掌握如此多的信息和知识，相同的学习 时间为何不同的人学习效率差别显著，首先要思考的问题就是：“人是如何学习的？”  这是一个多学科交叉问题，涉及到生物、心理、教育、社会、哲学、计算机等诸多领域，因此不同人和机构也从不同角度给予了不同的回答。

什么是学习？
当我们谈到学习的时候，我们通常会用这样的语言格式。
Someboday learn Someknowledge from Someone in Someway at Somewhere in order to do Something.
这其中也揭示了学习发生的故事情境。而其中的要素都会或多或少的影响学习。可以逐一分解，仔细研究。

最近读了几篇关于变构学习模型的文章，在这里，这里，还有这里。作者提出一个有趣的概念：概念体。受其启发，并结合我自己的理解，学习可以概括为认知主体 修正和调整自我行动范式从而提高在特定环境中工作的能力。认知主体具有自我概念体，这个概念体不仅是该主体解码外界信息的工具，同时也是学习需要改变的对 象。当概念体无法解码外界信号的时候，它会规避、忽略它，而只有面对他，学习才可能发生。

一、《How people learn》从教育学角度提出3大学习原则：
1 学习总是建立在原有的知识和经验基础上的，新的概念总是建立在前概念的基础上的，因此教学应该了解学会生的前概念，然后加以利用和区分进而建立新的概念。
2 一个人如若想在某个领域有所专长，他必须要掌握和理解大量的事实，并形成一定的理论框架进而理解相关的事实和概念，并对概念进行梳理从而更容易提取和应用。
3 元认知可以帮助人们调节自己的学习过程，确立学习目标，监督学习过程。

二、从学习的信息来源的角度，学习可以分为从直接经验学习和从间接经验学习两类。
直接经验比如通过试误学习，间接经验比如通过模仿学习、阅读学习等。

三、电脑做为机器，它的学习有哪些方式呢？
机械学习、通过传授学习、类比学习和通过事例学习。
现在机械学习和传授学习相对比较容易实现。但是通过类比和事例学些则相对于难于实现，其中一个关键的障碍就是概念，如何让电脑能够自主创造概念、使用概念和修正概念。

四、从学习对象和方法来看，学习又有以下可能方式：
在实践中学习，通过实践和尝试来学习。
社会化学习，即跟其他人学习。
两者相结合就是社会化实践学习。
阅读其他人的经验，阅读学习，然后反思和创新。

五、神经生物学视角下，学习是大脑神经网络的重构。
大 脑是学习的生理基础。大脑是由数目庞大的神经元连接成的具有信息识别、储存、加工、决策等功能的神经元网络。短期记忆改变的是突触的数目即神经元连接的强 度；长期记忆改变的是神经元网络的连接结构。学习的过程就是大脑神经网络在外部环境的刺激下自组织形成功能网络的过程。

信息素养的构成

发表于 2011-07-01 更新于 2013-01-11

下面是关于信息素养的一些摘录：
张情苇．信息素养与信息素养教育[J].电化教育研究，2001.(2)

美国国家信息素养论坛在1990年的年度报告中提出的信息素养人的标准：
(1)了解自己的信息需求；
(2)承认准确和完整的信息是制定明智决策的基础；
(3)能在信息需求的基础上系统阐述问题；
(4)具有识别潜在信息源的能力，能指定成功的检索策略；
(5)能检索信息源，包括能利用以计算机为基础的信息技术或其他技术；
(6)具有评价信息的能力；
(7)能为实际应用而对信息进行组织；
(8)具有将新信息结合到现存的知识体系中的能力；
(9)能采用批判性思维，利用信息并解决问题。

信息素养不仅包括利用信息工具和信息资源的能力，还包括获取识别信息、加工处理信息、传递创造信息的能力，更重要的是以独立自主学习的态度和方法、以批判精神以及强烈的社会责任感和参与意识，并将它们用于实际问题的解决和进行创新性思维的综合的信息能力。

信息素养主要由信息意识与信息伦理道德、信息知识以及信息能力三部分构成。
1.信息意识与信息伦理道德
信息意识是人们在信息活动中产生的认识、观念和需求的总和。信息意识主要包括：
(1)能认识到信息的重要作用，尊重知识、终身学习、勇于创新。
(2)对信息有积极的内在需求。
(3)对信息的敏感性和洞察力。
有信息责任感，能抵制信息污染，遵循一定的信息伦理与道德。
2．信息知识
信息知识是指一切与信息有关的理论、知识和方法。
传统知识+信息的基本知识+技术知识+外语。

3．信息能力

主要观点是围绕着问题和问题解决展开交流是最有效的。但是如何组织这样的交流呢？在线交流或者课堂交流中，作为一个教师应该如何回应学生呢？

The umbrella fact for all your responses: everyone in the class can read your postings.
Be a model of what you require of students in their postings.
Don’t be afraid to let personality and humor enter your responses.
While your response will be to one student, always include the whole class.
When applicable, use examples or experiences from your life in responding.
Create a bank of the best responses that you post for reuse in future classes.
Excerpted from “Teaching Online With Errol: Creating Effective Responses to Student Discussion Postings.” Online Classroom, April 2010: 6, 7.

  5. 记录和积累一些最好的回应，以备后用。
我们可以善用这些交流技巧，无论在课堂教学中还是在网络交流中。

如何让微格试讲的同伴交流研讨走向深入

发表于 2011-05-10 更新于 2013-01-11

今天杨同学跟我私下交流了一下自己的看法，ta说多次试讲同一个内容，大家都疲惫了，交流的内容也总是重复。微格课程中确实存在一个较为严重的问题，那就是大家都是新手，生活经历以及知识范畴相同度很高，对于教学的认识层次也相差不大，这样需要就造成对于同一个问题相同的声音太多，而不同的观点或者不同角度的看法就较少，一句话，同感有余，而异质交流不足。我觉得这个问题从以下几个方面来解决：

  1. 鼓励微格学员打开自己的思想束缚，认真倾听别人的意见和见解，发现不同之处。很多时候，并不是表面上看到的那样，大家观点都一致，而是你没有发现其中的不同，或者有些不同的声音被忽略掉了，被过滤掉了，甚至没有来得及发出。认真的倾听，去发现别人的可取之处尤为重要。
  2. 引入异质成员，比如请一些一线教师参与交流给他们指点，我觉得可以把他们实习的指导老师邀请进来，或者让他们走出小社区，走进大社区，比如好看簿上的相关社区，以及生物教学茶吧等社区。这样，就会有不同的声音进来，相互启发之处会增多。
  3. 除了对某个内容反复试讲之外，还应该带着研究的性质对这部分的内容的教学进行研究。研究的过程包括查阅相关教学设计的文献，查找相关专家教师的视频案例并一起观看、分析、模仿、借鉴，甚至可以尝试把自己的教学设计化为真实的教学过程，比如与一线教师合作，或者直接走进中学课堂。人说读万卷书，行万里路，说的就是知识来自阅读，实践出真知。

    * 2- 不显示任何的关闭按钮
    * 3- 在当前所有标签页的最后显示一个关闭按钮

从宣讲到对话——教学的发生了哪些改变？

讲座式教学在高校中很流行，教师根据讲义宣讲特定的内容，然后考试也考察这些内容。有的老师讲的很精彩，例子贴近学生生活，让学生在平常中发现不平常，深入浅出，发人深思，令人反省。有的老师讲起来则显得干涩，抽象，缺乏互动交流，令人昏昏欲睡。这当然于老师的教学功底、教学经验、学科知识涵养有关，要想讲的好绝不是一朝一夕可以成就的。换句话说，一步到位的成果教学是极为困难的。

为什么呢？因为学科知识的深入理解需要时间，教学经验的积累也需要时间，问题的优化需要时间，例子的筛选更需要时间，了解学生当然也需要时间。如何达成这些必要条件呢，答案是教学实践，在实践中尝试、摸索、积累、反思。

我教学理论说教的系统的备课，如何进行内容分析、学生分析、目标设定、教学过程设计，说的头头是道，但是是不是学了这些理论就能一次性达到成功教学呢？答案是否定的，这些理论知识会有一定的作用，但是根本不能保证一步到位的成功教学，因为教学的本质不是宣讲，而是对话。宣讲是可以完全系统预设的，但是对话无法完全预设，就好像打乒乓球一样，你发出去的球，学生可能接得住也可能接不住，即使接到了，打回来的球的方向你也无法完全确定，只有当你和学生在这个比赛中充分熟悉之后，你才能有更好的预判。教学也是如此，与其期望由教育理论构造出一种似乎可以一步到位的教学设计步骤，倒不如帮助准教师学会对话，学会教学反思，在实践的过程中快速成长。

这是《有效的学生评价》中的一个观点。

其中有很多原因了，最常见的，老师们会说高考就是这种形式啊，平时考试当然要为高考做准备。另外一个重要原因是老师已经习惯了找一些题目做一次测试，想要换一种评价方式，还真不知道如何操作了。可见除了应试教育的原因，老师自身关于评价的知识和技能储备也是个问题。其实最根本的还是观念的转变，观念不变，心态不开放，这种陋习还是一直延续下去。到最后，学生讨厌考试，教师也会产生考试倦怠，而测评的真正目的却没有达成。

  • 激活学生的先前知识和经验
  • 运用视觉形象来弥补和支持书面材料的学习
  • 运用多好中途径帮互助学生表达思想，即让他们们运用不同的学习方式和语言
  • 用图来演示各种观点之间的关系
  • 用视觉形象对物质材料进行概括
  • 评价团队评价活动帮助那些英语水平较差的学生。

每当我们要进行一次测评时，我们都应该好好想一想评价的目的是什么，评价哪些方面，应该采用怎样的评价方式最为合适？而不是被动评价，为了评价而评价。

在高中生物学中有一个概念叫染色体，所谓染色体组就是指一组形态大小各功能不相同、但是包含了本物种生长、发育和繁殖所需的所有遗传信息的非同源染色体。这个概念是理解二倍体、多倍体以及单倍体的基础。人有46条染色体，实际上是两套，从1号一直编到22号，每号都有两条，在加上2条性染色体，刚好23对。所以人的一个染色体组包含23条染色体，但是学生总是不容理解，学生总是倾向于把形态大小相同的两条染色体划为一组。这可如何是好？ 于是我在讲解了染色体组概念的定义和实例之后，我打了一个比方，染色体组就好象一个篮球队，每个队伍一般都要有小前锋、大前锋、中锋、得分后卫、组织后卫等，每个人都有自己的特殊技能，共同组成一个整体。你不能组一个队伍全都是中锋，哪这球就没法打了。染色体组也一样，每条染色体都是不可缺少，但是它们彼此又各不相同，他们共同包含了一个遗传信息整体。

如何在ppt中插入flash动画

发表于 2011-02-27 更新于 2013-01-11

我在教学中遇到这个问题，教室里的电脑无法播放flash，用IE虽然可以播放，但是切换窗口容易损失学生的注意力，并且也浪费时间。所以我要把flash插入到ppt中播放。我试过直接把文件拖放到ppt的特定页面中，结果是建立了一个链接，这样还是要切换窗口，也容易转移注意力，这不是我想要的，我试过插入对象，但是一下子弹出了一个vb窗口，把我给吓住了。于是解决问题，还是要搜索，结果搜索到下面的方法，尝试了一下，结果只要在movie字段后面输入flash地址即可，问题解决。

利用控件
　　这种方法是将动画作为一个控件插入到PowerPoint中去，该方式的特点是它的窗口大小在设计时就固定下来，设定的方框的大小就是在放映时动画窗 口的大小。当鼠标在Flash播放窗口中时，响应Flash的鼠标事件，当鼠标在Flash窗口外时，响应PowerPoint的鼠标事件，很容易控制。
　　1. 运行PowerPoint程序，打开要插入动画的幻灯片。
　　2. 单击菜单中的“视图”选项，在下拉菜单中选择“工具栏”的“控件工具箱”，再从下拉菜单中选择[其他控件]按钮。
　　3. 在随后打开的控件选项界面中，选择“Shockwave FlashObject”选项，出现“十”字光标，再将该光标移动到PowerPoint的编辑区域中，画出适合大小的矩形区域，也就是播放动画的区域，就会出现一个方框。
　　4. 双击这个框，出现VB界面，其中的属性窗口中有个“自定义”属性，点自定义旁边的三点，出现属性页。在“MOVIE URL”中输入Flash动画的完整地址，按[确定]即可。要注意的是输入的路径中必须完整地填写后缀名。

元宵节上课遇突发事件

发表于 2011-02-17 更新于 2013-01-11

今天是我回学校上的第一节课，前面5节课都让L老师帮忙上了。这节课讲的是有丝分裂，提前准备了幻灯片、相关动画和视频以及教学构想，但是到了课堂上，班级里的电脑怎么都不能登陆，于是昨天准备的所有电子资料都不能用了，不能看图片，也没有电子表格和动画视频了，教学突然又回到了一支粉笔一本书的时代。这确实给我提出了不小的挑战，但是我镇定了一下，还是用一张嘴和一支粉笔和一本书把这节课的内容讲完了。当然形象直观肯定大打折扣，但是对老师临场应变能力却有锻炼。

今天LBY老师说上学期学生的成绩考的不错，问我有没有好的经验介绍一下，我说我还有好好的反思。确实，我还是感觉到自己在教学上做的还没有达到自己的要求，游刃有余，并实现自己对教学的观点和认识。不过我也发现，反思和记录对于教师的成长地区有好处，我现在就特想把与教学有关的信息，按照教学内容全部记录下来，这些是以后改善教学的基础，也是教学研究珍贵的记录。那么，对于新教师，教某个内容应该记录哪些东西呢？
教学遇到的问题可以记录；（比如，今天下课后一个女同学问我为什么细胞要分裂，为什么人会有意识，为什么各种元素可以组成生命，我尝试意义解答，花了很上时间，后来也黔驴技穷，于是我再思考对于这样的追问，教师应该怎样应对？）
学生的典型错误可以记录；（比如，学生经常把有性生殖和卵式生殖混淆，不知道什么地方填哪一个，因此以后教学对于这两个概念要注意区分和辨析）
此外好的教学资源更可以记；好的教学构想可以记；好的问题设计可以记；学生对问题的反应情况也可以记。

试想如果你把这些东西都进行了记录和反思，那么下一次你的教学设计和实施就会轻松很多，但同时也会改善和提高很多，应为你吸取了前面的经验教训。其实，教师的学习其实就是在不断犯错误和纠正错误的过程中发生的。如果一个老师能够把自己对某个教学内容的所有相关东西进行记录和链接，那么这个资源不仅对教师自己非常有帮助，而且对于其他老师也非常有益。
