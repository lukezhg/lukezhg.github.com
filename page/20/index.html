所有知识始于自知

发表于 2017-03-24 更新于 2013-01-11

“Self-knowledge is the beginning of all knowledge,” writes C. Roland Christensen

自我阻碍是每个人都要经历的一个砍，勇敢的走过去，才能继续前行。
今天讲了学习理论，主要介绍了行为主义、认知主义、建构主义和连接主义四个流派的学习理论。介绍了相关的趣味实验，但是还是没有找到贴近学生生活的鲜活的例子，也没有设计出恰当的有挑战性的问题。所以一个理论一个理论介绍下来，还是觉得有些枯燥，不够活，不够精彩。自己的演讲基本功，比如组织语言能力、语调变化、讲故事的能力都还要提高。不过鉴于今天的状态（感冒 嗓子疼），能坚持把课上完，并大体上把内容讲完，也就知足了。

课后，z老师提出了建议也非常诚恳，我应该在自己所举的例子、设计的问题、所讲的任何内容都应该试图寻找它与中学 生物 教学的关系在哪，什么样的知识是学生需要的，什么样的问题是他们会遇到的。设身处地的想和设计。

中学生物学设计孵化器

发表于 2017-03-24 更新于 2013-01-11

对于生物学教师来讲，如何提高备课的效率和质量呢？
备课应该遵照一个程序，符合教育教学的基本规律，充分利用各种教学资源。那么，加入有一个现成的页面，能够把备课所需的相关资源按照一定的教学理念有机的整合在一起，岂不是就可以让教师的备课事半功倍吗。
于是，我就新建了这样的一个意在提高生物教师备课效率的wiki页面，我将之称为“中学生物学设计孵化器”，当然还有很多方面需要完善，先自己试用，在用的过程加以完善，相信也会对其他高中生物教师的备课有点帮助。

让“帮助”传递下去

发表于 2017-03-24 更新于 2013-01-11

一个多月前，本人不慎丢失了自己钱包，连同自己身份证、银行卡、电影卷（50元）、80元左右的现金，一起丢失。估计是丢在小区的路上，本来我是想到取款机去取钱的，结果到了钱包没了，于是马上原路返回去找，没找到。会议当时的情形，有三个保安正面走过我途径的路上，事后我也向他们问起是否有人捡到，他们答，即使捡到现金也可能没有了，我说不要紧，能把那么多卡找回来就好。一个星期过去，未果。

于是只好补办身份证，先是挂失，然后一个月后去正是申请补办。这时一个陌生人让我感觉还是好人多。事情是这样的，我拿着凭证来到派出所，但是发现补办身份证要40元，我愿意为最多也就30元，身上钱带的不多，结果搜遍全身，也只有39元，于是我就被难倒了。因为出去取钱的话，可能要走很远，而且也赶不上班车了，于是我跟民警说了情况，民警阿姨有些犹豫，这时我后边的一个女人（她是领女儿办身份证的）从口袋里拿出一元钱，我很尴尬，但是还是接受了她的好心帮助。我知道这钱很难还给她了，而且她也并不指望我还，但是我却要非常感谢她。虽然不能还钱了，但是其她感谢的方式还是有的，就是在自己力所能及的请跨把类似的帮助传递给别人，让社会更加友善！

元宵节上课遇突发事件

发表于 2017-03-24 更新于 2013-01-11

今天是我回学校上的第一节课，前面5节课都让L老师帮忙上了。这节课讲的是有丝分裂，提前准备了幻灯片、相关动画和视频以及教学构想，但是到了课堂上，班级里的电脑怎么都不能登陆，于是昨天准备的所有电子资料都不能用了，不能看图片，也没有电子表格和动画视频了，教学突然又回到了一支粉笔一本书的时代。这确实给我提出了不小的挑战，但是我镇定了一下，还是用一张嘴和一支粉笔和一本书把这节课的内容讲完了。当然形象直观肯定大打折扣，但是对老师临场应变能力却有锻炼。

今天LBY老师说上学期学生的成绩考的不错，问我有没有好的经验介绍一下，我说我还有好好的反思。确实，我还是感觉到自己在教学上做的还没有达到自己的要求，游刃有余，并实现自己对教学的观点和认识。不过我也发现，反思和记录对于教师的成长地区有好处，我现在就特想把与教学有关的信息，按照教学内容全部记录下来，这些是以后改善教学的基础，也是教学研究珍贵的记录。那么，对于新教师，教某个内容应该记录哪些东西呢？
教学遇到的问题可以记录；（比如，今天下课后一个女同学问我为什么细胞要分裂，为什么人会有意识，为什么各种元素可以组成生命，我尝试意义解答，花了很上时间，后来也黔驴技穷，于是我再思考对于这样的追问，教师应该怎样应对？）
学生的典型错误可以记录；（比如，学生经常把有性生殖和卵式生殖混淆，不知道什么地方填哪一个，因此以后教学对于这两个概念要注意区分和辨析）
此外好的教学资源更可以记；好的教学构想可以记；好的问题设计可以记；学生对问题的反应情况也可以记。

试想如果你把这些东西都进行了记录和反思，那么下一次你的教学设计和实施就会轻松很多，但同时也会改善和提高很多，应为你吸取了前面的经验教训。其实，教师的学习其实就是在不断犯错误和纠正错误的过程中发生的。如果一个老师能够把自己对某个教学内容的所有相关东西进行记录和链接，那么这个资源不仅对教师自己非常有帮助，而且对于其他老师也非常有益。

如何在ppt中插入flash动画

发表于 2017-03-24 更新于 2013-01-11

我在教学中遇到这个问题，教室里的电脑无法播放flash，用IE虽然可以播放，但是切换窗口容易损失学生的注意力，并且也浪费时间。所以我要把flash插入到ppt中播放。我试过直接把文件拖放到ppt的特定页面中，结果是建立了一个链接，这样还是要切换窗口，也容易转移注意力，这不是我想要的，我试过插入对象，但是一下子弹出了一个vb窗口，把我给吓住了。于是解决问题，还是要搜索，结果搜索到下面的方法，尝试了一下，结果只要在movie字段后面输入flash地址即可，问题解决。

利用控件
　　这种方法是将动画作为一个控件插入到PowerPoint中去，该方式的特点是它的窗口大小在设计时就固定下来，设定的方框的大小就是在放映时动画窗 口的大小。当鼠标在Flash播放窗口中时，响应Flash的鼠标事件，当鼠标在Flash窗口外时，响应PowerPoint的鼠标事件，很容易控制。
　　1. 运行PowerPoint程序，打开要插入动画的幻灯片。
　　2. 单击菜单中的“视图”选项，在下拉菜单中选择“工具栏”的“控件工具箱”，再从下拉菜单中选择[其他控件]按钮。
　　3. 在随后打开的控件选项界面中，选择“Shockwave FlashObject”选项，出现“十”字光标，再将该光标移动到PowerPoint的编辑区域中，画出适合大小的矩形区域，也就是播放动画的区域，就会出现一个方框。
　　4. 双击这个框，出现VB界面，其中的属性窗口中有个“自定义”属性，点自定义旁边的三点，出现属性页。在“MOVIE URL”中输入Flash动画的完整地址，按[确定]即可。要注意的是输入的路径中必须完整地填写后缀名。
以上文字来自：http://www.pconline.com.cn/pcedu/soft/office/ppt/0312/253098.html
