用积极主动7个习惯反思自己一年的学习

发表于 2009-12-26 更新于 2013-01-11

这一年的重头戏是论文工作，我做的方向是生物教师的学科教学知识发展，从选定题目到现在还没有完成数据的收集老看，自己的这7个习惯执行的还不好。自己另外欠缺的就是周期性的总结，我没有总结的习惯，但是却又发散思维的特征，常常想着想着就可以冒出一些新的想法，但是这些想法又比较模糊，不容表达清楚，我也对这些想法做了记录，但是还是难于派上用场，究其原因就是刚才提到的，缺乏总结的习惯。其实总结是一种反思的方式，总结是痛苦的过程，因为总结一般是对已发生的事情的回顾和剖析，或者说洞察，发散可以说是在做加法，而总结则是要做减法，凝练，概括，检讨，回归现实，所以对于我这样的总是喜欢有些幻想的人来说，总结确实是非常痛苦的事儿。

总结和反思自己也需要一个框架，七个习惯就提供了一种可能的框架。


从这张图来看，自己欠缺之处在于“要事第一”自我控制 以及“知彼知己”的沟通能力。

要事第一说起来简单，但是操作起来对我很困难。因为有的时候要事也是难事，难事是不要做的，所以迟迟做不下去，而且在生活中也很容易被一些不要紧的是干扰。要事第一的难处在于合理拒绝干扰，强横的法则执行意识和监控能力。

沟通也是同样重要和困难。实际上，人与人的沟通是社会化组织形成的根本途径。沟通时双向建构的过程，沟通也有很多技巧，此外沟通也要消耗时间成本。在论文实施的过程中，有效的与教师进行沟通成了我论文推荐的障碍，直到现在还没有很好的解决。一线教师非常忙，他们视乎只为了眼前的课时疲于奔命，而对于其他无暇顾及，用他们自己的话说“课时太多，会太多，没办法”。我真想努力帮助他们改变这种现状，但是我在技术和方法上的努力视乎并没有立竿见影的效果。也许这就是技术与教学生活整合的困难之所在吧。

关于技术如何与教师教学工作和生活整合，是我有一个想关注的目标。对于这个问题，欢迎大家评论或者用博文讨论。今天先总结到这里。

教学中的几个重要时间点

发表于 2009-12-06 更新于 2013-01-11

3分钟
教师提问后，如果这是一个只得让学生花时间思考的问题，那么也应该给学生思考的时间。这个时间叫做wait time，通常要有3分钟左右。 而那些试探性的问题，学生能够张口就来的不需要思考的问题，设置的目的是检查学生的状态和注意力，对于发展学生的迁移能力和问题解决能力作用不大。个人觉得一堂课至少要给一个3分钟思考的重要问题，教师不能只是给出一些没有挑战性的问题，或者给出挑战性的问题，却不给足够的时间，急于讲解。论语中说，不愤不启，不悱不发，要设置一些好问题，并给出一点时间让学生进入角色，达到愤和悱的状态，然后才好启，才好发。急于求成，并不能达到很好的效果。

20分钟
这个数字可能不一定确切，但是大概是这个时间。在一节课中，一班学生的注意力或者说认知高潮只能维持20分钟左右，如果教师一个劲儿的讲，并且不采用其他一些策略吸引学生参与的话，课堂的后20分钟学生的学习效果就会大打折扣。20分钟也应该是教师理想的精讲时间的上限，40分钟或45分钟的满堂讲授，不仅教师很累，学生也会很累。那么是不是教师也应该考虑给学生多一些时间表演呢，教师不能老是抢戏，要知道独角戏很难演的。

课前5分钟
我还发现一些有经验的老师，会至少提前5分钟进入课堂，进行课堂准备。比如把课件放到电脑里，并打到屏幕上，交代课代表一些关于作业的事情，解答学生一些疑难，并且在正式打铃前1-2分钟，就开始维护课堂记录，让学生回到座位，拿出书本，安静下来。这也是一种争取课堂时间的方法。否则这些准备工作就可以占用掉5分钟的课上时间，如果每节课都是这样，一个月就相当于少上了2节课。

从“不愤不启，不悱不发”到概念教学

发表于 2009-12-05 更新于 2013-01-11

这句话出自《论语》，下面引一段解释。

在《论语》“述而”篇中，孔子又说：“不愤不启，不悱不发。举一隅不以三隅反，则不复也。”对此，朱熹的解读是：“愤者，心求通而未得之意。悱者，口欲言而未能之貌。启，谓开其意。发，谓达其辞。物之有四隅者，举一可知其三。反者，还以相证之义。复，再告也。”
来自：http://blog.sina.com.cn/s/blog_48b0011f0100bykg.html
愤可以对应为认知冲突，百思不得其解的状态
悱可以对应为好像有想法但是缺乏表达的语言
启则是为了解决认知冲突而进入科学概念，而使学生豁然开朗。
发是给学生提供表达的词汇帮助其达其辞，而让学生清晰表述。

同时这句话涉及到了教学时机的把握的问题，只有当学生愤时方启，悱时才发，否者教师光顾着自己启发，学生还没有进入相应的状态，那么也必然是教师自己瞎忙活罢了。

因此，优秀的教师，要能相仿设法帮助学生达到这些状态，并且及时应对，才会达成较好的教学效果。

教学就是建造滑梯

发表于 2009-12-04 更新于 2013-01-11



为了建构好这个滑梯，教师要知道学生的位置，也就是学生原有的起点，接下来才能在学生起点与目标之间建立小步的学习阶梯。从一个阶梯到另一个阶梯，教师可以通过问题来引导。问题是牵引着学生一步一步的爬梯子，并最终达到顶端，爬楼梯是一个辛苦的过程，辛苦之后学生应该获得一些回报，也就是要给与鼓励。那么从高处往下滑的游戏，就是一种很好的鼓励方式，也就是让学生应用新学的知识，使得以前的迷惑豁然开朗。只有让学生很享受，学生才会更愿意爬另外一个阶梯。

但是看看上面的图片，这个滑梯还是挺危险的，学生如果不小心摔坏下来怎么办。 所以教师还需要在下面铺一层海面，或者做一些保护网，这样学生才能在安全的环境下去探索和跳跃。教学也一样，老师要想方设法给学生营造一个安全的表达观点或者看法的环境，要容许学生犯错误，并且鼓励学生互相帮助和协助，而不是嘲笑和愚弄。

量子学习包括6种策略：

引入（Enroll）—教师用行动抓住学生的兴趣、好奇心和注意力

感知（Experience）—创造或者引导出一种共同的经验或者是与所有学习者都相关的常识性知识。在学习标签之前让学生先获得感受，进而在这个基础上建构新的内容。

精致（Learn & Label）—逐步展示并定义主要内容。学生学习新名词，思维技巧和学术策略。学生再原有框架上增添新内容。

应用（Demonstrate）—给学生提供展示和应用新学内容的机会。

回顾（Review and Reflect）—用多种有效的、多感官的回顾策略，让学生通过反思来巩固新学内容

庆祝（Celebration）—承认学习结果。这可以巩固内容并增加一种完成的感觉。



利用网络社区学习的优势和劣势

发表于 2009-11-05 更新于 2013-01-11

一、网络社区学习的优势
1、跳出身份意识的束缚
最大的优势，就是我可以问任何自己想知道的问题，哪怕是常识性的问题或者是幼稚的问题，我不需要再刻意估计自己的身份，也不需要装作精通所有事情的专家。我的目标是学习和思考，而不是权威。有的时候，我也可以自问自答，至少我自己是我提问的问题的忠实关注者。

2、突破时空的约束
还有一个优势就是可以突破空间和时间的限制，与世界各地的具有同样兴趣的专业人士进行对话，对话即是人与人相互了解的过程和建立人际网络的过程，同样也是学习的过程，是建立内部知识网络的过程。如果你想有所收获，你就要深入的卷入到你自己关心的问题的对话浪潮中。

3、随时对地
还有个特点，也许和第二点有些重复，那就是它的容易获得性。你随时随地都可以发起一个对话，而不像现实世界，你必须要找到专家或者朋友才能开始你的话题。

4、记录——反思——质的飞跃
再有一个好处就是记录性，因为我们是用文字来进行交流的，对话的过程，也就是记录的过程，而记录的作用会在不断的积累过程中发生质的变化。当你对一个问题进行了较为广泛对话和较为深入的思考之后，你就会对这个问题有一个新的认识。这个认识已经记录在你的对话之中了，你只需要在对过去的对话进行一下梳理，总结一下要点，就可以较为容易融入到你自己的知识体系当中。而记录的另外一个作用是，回顾和反思。对话记录了你思想发展的过程，反思可以帮助产生质的飞跃。 而这样的飞跃产生的知识，又可以外化为文章或者博文进行分享和传播。从而提高对话的质量，为社区学习群体增添更多的价值。

小结
对话是对学习社区贡献的重要方式，也最学习社区最基本的实践活动形式。有了对话，才有了成员之间的认识和理解以及信任，才有了集体协作共同完成项目的可能。而群体协作，特别是自发的有组织的协作，其威力是巨大的，它可以完成个体不可能完成的任务。比如在3个月写一本书或教程，翻译一部巨作，甚至像Wikipedia一样，写一个无所不包的百科全书。

二、网络社区学习的劣势

  • 网络沟通不如面对面沟通亲切，传递的信息有限，而且具有一定的危险性。
  • 学习使用网络、善用网络和计算机需要一个过程，具有一定的技术门槛
  • 网络资源无限丰富，无限链接，筛选成本高，且容易让用户迷失方向和自我
  • 网络的信息具有一定的不安全性，因为信息接收者身份不明，可能会滥用和盗用

从搜索文化到转发文化

发表于 2009-11-04 更新于 2013-01-11

google,百度等搜索公司都是在搜索文化中发展起来，兴盛起来的。而随着twitter为首的微博客信息链的架起，一种新型的文化正向我们走来，那就是转发文化。搜索文化的特点是内容静止，而用户更具自己的需要搜索直接相关的信息。转发文化则是人是静止的，内容沿着人与人搭建的信息链自主传递而来，比如twitter,google reader等都有类似的特征。虽然转发行为中，转发者并不一定增加新的信息，但是却起到了智能筛选、匹配的类似智能路由器的作用，这个过程，人的智能已经融入到了信息投递的过程。

如果经过恰当的搭配和处理，将形成一个人脑和电脑高度整合的智能处理网络，这个网络不仅能处理和解决问题、投递信息，而且也将在企业管理、教育教学、科技创新等领域中发挥越来越大的作用。Technorati 标签: 转发文化, 搜索文化, twitter, google, baidu

搜索信息解决问题的步骤和方法

发表于 2009-10-19 更新于 2013-01-11

1、首先是表述问题
遇到了问题，不见得会表述问题，表述问题要尽量附带一些背景性信息，并且不能或略问题或者现象的关键点。另外，也也要考虑表述给谁？当然首选的就是表述给搜索引擎，原因很简单，因为google等搜索引擎它们会24小时恭候你的使用。下面的步骤如何选择搜索引擎，如何表述给搜索引擎。

2、搜索相关信息和资源
不同的问题，可以选择不同的搜索引擎，比如如果你要搜索一个学术问题或者文章，你应该首选scholar；而如果你想搜索一个生活常识或者古诗词，则选择百度百科可能会更好。如果你想下载一个软件，可以选择sogou来搜索。

选择好了引擎，还有要把问题表述给引擎，你应该从自己问题中找出关键点或关键词，或者你先可以把自己的想法全句拿去搜索，然后依据搜到的结果多少和相关度进一步对问题和词句进行修改和删减等。要有一定耐性，并摸索一些技巧，其中懂一些基本的搜索语法和关键词是必要的，参google高级搜索符的完整列表。

3、求助专家
搜索引擎并不能解决所有的的问题，比如你在python编程的过程中有一个功能无法用恰当的代码实现，你搜索不到相关的信息，因为程序员很少，而关注这个功能的程序员就更少了，何况每个程序员使用的编程语言也千差万别。因此，这个时候你就需要找到python编程的专家，那么到哪里去找呢，当然是去坛子里找。这个时候你的搜索对像就变成了python的网络论坛，找到相应的好的论坛之后，再发帖求救，并及时关注和恢复，通过这种方式可以大大增大你解决问题的可能性。

4、建立环境
我们开始的时候，为了解决某个领域的问题，经常会要用搜索引擎，因为它包罗万象，只要你有动力，有时间，绝大多数的信息都是可以搜到的。但是搜索的相关较差，需要花时间进行赛选。另外由于遗忘等因素，也会造成重复性的工作和低效率。接下来，我们需要限制自己的搜素领域，从而提高命中率，我们可以建立某个范围的搜索引擎来节约时间。当遇到难题时，由于互联网上不一定有问题的答案，答案在专家的头脑里，所以我们需要找到专家。实事上，建立一个个人专家求助的人际圈子很重要，它是一种无形的资源，有了他们在身边，一些你还没有遇到的问题，都可以被他们解决，原因在于你在无形中学习到了他们提供的技巧和习惯。那么如何让你以后遇到问题，能最快速的解决呢。答案是建立一个领域学习的环境，比如茶吧这个生物教师学习社区，它就是这样一个领域学习环境。如果你有关于生物教学的困惑和需要，那么你应该入住这个社区，通过与专家和高手建立联系来组建你个人的专家求助热线。

5、如果我还想更快的的解决问题，该怎么办？
答案是，自己成为专家。
其实前面四个步骤已经为你成为专家打下了基础，特别是一个专业学习环境的建立，它将为你成长为一个专家提供各个方面的支持。比如教程、技术技巧、思想、信息资源、人际关系等。当你自己成为一个专家以后，你不在需要花那么多的时间搜索和找人，因为你搜索到的信息很多是你写的，很多人是你认识的，而且如果一个问题如果你解决不了，别人也很难解决。这个时候不是你找别人，而是别人找你。不是你找答案或者别人提供答案，而是你研究并提供解决方案。
